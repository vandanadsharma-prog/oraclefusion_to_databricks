
  # Build app from file

  This is a code bundle for Build app from file. The original project is available at https://www.figma.com/design/o2Vv2tMEb2bzLqveSe5UjW/Build-app-from-file.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Backend (FastAPI)

  A FastAPI backend is available in `backend/` to power the “Run Pipeline” button.
  See `backend/README.md`.

  For a step-by-step local setup (Oracle 21c + GoldenGate + frontend + backend), see `LOCAL_SETUP.md`.
